# Changelog

All notable changes to Tiny Windows Shutdown Scheduler will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Planned Features
- Dark mode support
- Keyboard shortcuts
- Custom preset timers
- History of recent actions
- Sound notifications

## [1.0.0] - 2026-05-27

### Added
- Initial release
- Schedule shutdown, restart, sleep, hibernate, lock, and log off
- Preset timers (15 min, 30 min, 1 hour, 2 hours, tonight)
- Custom timer with hours and minutes
- Live countdown display
- Confirmation dialog before executing actions
- Cancel button for scheduled actions
- Portable executable (no installer needed)
- MIT License

### Features
- Tiny native Windows executable (~29 KB)
- Simple, intuitive UI
- Windows native commands for reliability
- No Python, no Electron, no Node.js dependencies
- No telemetry or data collection
- Open source and customizable
